Tutorial 4: Video Call 1 to 1 with WebRTC

This web application consists on an one to one video call using WebRTC. In other words, this application is a video phone. The Media Pipeline is composed by two interconnected WebRtcEndpoints. To run this demo follow these steps:

  1. Open this page in a WebRTC-compliant browser (Chrome, Firefox).
  2. Type a nick in the field Name and click on Register.
  3. In a different machine (or a different tab in the same browser) follow the same procedure to register another user.
  4. Type the name of the user to be called in the field Peer and click Call.
  5. Grant access to the camera and microphone for both users. When the connection is negotiated, communication should start.
  6. The called user must accept the incoming call (by a confirmation dialog).
  7. Click on Stop to finish the communication.
